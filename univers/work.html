001 : News Website

This project was my final assignment in the Course of mobile web development. I created a website for news articles. I encountered several challenges during the development process, including designing an intuitive user interface, implementing responsive design for various devices, and optimizing the website for fast loading times.

To address these challenges, I conducted thorough research on modern web design principles and utilized frameworks like Bootstrap to streamline the UI development process. I also employed CSS media queries to ensure the website looked and functioned seamlessly across different screen sizes.

One of the main difficulties I faced was integrating dynamic content from a news API while maintaining the website's performance. To tackle this, I implemented server-side caching and optimized database queries to minimize load times.

Overall, this project helped me enhance my skills in web development, particularly in frontend design, backend integration, and performance optimization.
